# Overview

The Filipino Emotion Recognition Model (FERM) is a planned transformer-based Natural Language Processing (NLP) model for CTRL4 Chatbot.

Its purpose is to recognize emotions expressed in Filipino, Tagalog, and eventually code-switched conversations to improve emotional understanding during chatbot interactions.

The model has not yet been developed and currently exists as part of the project's future roadmap.

# Intended Use

The model is planned to support:

- Filipino emotion detection
- Tagalog emotion recognition
- Emotion-aware prompt construction
- Student guidance conversations
- Multilingual chatbot interactions

---

# Not Intended For

The model is not intended for:

- Mental health diagnosis
- Clinical decision-making
- Psychological assessment
- Medical recommendations
- Replacing licensed guidance counselors
