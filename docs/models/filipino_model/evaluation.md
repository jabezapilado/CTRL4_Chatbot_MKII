# Overview

This document will present the evaluation results of the Filipino Emotion Recognition Model (FERM) once development is completed.

The model is planned to support emotion recognition for Filipino and Tagalog student conversations, improving the chatbot's understanding of multilingual and code-switched communication commonly used within the university.

At present, the model has **not yet been trained or evaluated**.
