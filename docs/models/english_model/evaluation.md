# Overview

This document presents the evaluation results of the English Emotion Recognition Model used in CTRL4 Chatbot MK II.

The model was evaluated using a held-out test dataset after fine-tuning on the merged English emotion dataset. Within the chatbot, the model serves as the emotion detection component of the AI pipeline, providing contextual information that helps the Prompt Builder and Large Language Model generate more empathetic and context-aware responses.

# Model Strengths

The English Emotion Recognition Model demonstrated several strengths during evaluation.

- High confidence for clearly expressed emotions.
- Fast inference suitable for real-time chatbot interactions.
- Balanced performance across all supported emotion classes.
- Effective separation of positive, neutral, and negative emotional states.
- Seamless integration with the CTRL4 AI pipeline.

---

# Current Limitations

The current version of the model has several known limitations.

- Supports English text only.
- Does not interpret emojis.
- Limited understanding of internet slang.
- Limited support for code-switched conversations.
- Predicts emotion independently for each message.
- Does not yet support Filipino or Taglish emotion recognition.

---

# Intended Application

The English Emotion Recognition Model is designed to identify the emotional state expressed in a student's message before response generation.

Its predictions are used to:

- Provide emotional context for prompt construction.
- Guide empathetic response generation.
- Support future counselor escalation features.
- Improve the overall conversational experience.
- Enhance context-aware interactions within the chatbot.

# Summary

The English Emotion Recognition Model demonstrated reliable performance for multi-class emotion classification, achieving approximately 79.5% overall accuracy with balanced precision, recall, and F1-score.

Within CTRL4 Chatbot MK II, the model serves as the emotion detection component of the AI pipeline. Its predictions provide contextual information for prompt construction, allowing the configured large language model to generate responses that are more empathetic and appropriate to the student's emotional state.

Although the chatbot's final responses are generated by a large language model, the emotion classifier remains an essential component that improves emotional awareness and provides a strong foundation for future enhancements such as conversation-level emotion tracking, multilingual support, and intelligent risk assessment.
